*** Pricipais idéias levantadas:

- O objetivo principal do blog seria apresentar de maneira prática
	sugestões de atividades fora das telas. 
- Os itens seriam apresentados de maneira sucinta para que os
	tutores também não tenham que "gastar" tempo nas telas,
	sendo assim um exemplo, além de muitas vezes não terem
	muito tempo disponível para isso.
- Os usuários que se cadastrarem terão acesso a um fórum, 		ou "comunidade", para que possam compartilhar 	experiências. O fórum teria apenas um moderador para 	garantir que as regras sejam respeitadas. O moderador
	seria um administrador.

	
- Página principal
	- Apresentar as atividades como a visualização de
		produtos em uma loja, com ícones de download,
		favorito e avaliação (estrelas)
	- Ao clicar na atividade poderá fazer download do arquivo
		contendo o passo a passo (como uma receita)
		e também uma explicação "teórica" simplificada
		do porque a atividade é boa para a criança
	- A tela apresentará um filtro, onde o tutor poderá
		informar o tempo disponível para realizar a 
		atividade, a idade da criança e o tipo de
		atividade desejado.
	- As atividades poderão ser ordenadas pela data de criação
		(data de postagem) ou tempo de execução.
	- A tela principal apresentará uma pequena área para
		o usuário informar email e/ou telefone para 
		receber as últimas novidades.
	- Ícone de Login

- Página Insitucional
	- Página onde apresentamos quem desenvolveu o blog e
		o motivo de ser desenvolvida.

- Página de Contato
	- Página onde o usuário poderá entrar em contato com os
		criadores para dar sugestões, reclamações ou
		solicitações.

- Página de Cadastro do Usuário

- Página de indicações 
	- Indicar alguns parques, museus, exposições etc que
		são mais conhecidos no país.

- Área do Usuário
	- Página de Dados Pessoais
	- Página de Downloads
	- Página de Favoritos
	- Acesso ao Fórum de Tutores

- Área do Administrador
	- Página para o cadastro dos posts
	- Página para gerenciar os subscribers
	- Página para gerenciar os usuários cadastrados
